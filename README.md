# Sistema de Clínica em Python

Projeto desenvolvido para gerenciamento de uma clínica médica feito por mim, integrando banco de dados MySql e Python, com funcinalidades de cadastro, exclusão, agendamento, e alteração.

## Funcionalidades
- Cadastro de clientes
- Cadastro de médicos
- Agendamento de consultas
- Alteração e exclusão de dados

## Tecnologias
- Python
- MySQL
- HeidiSQL
